# FlutterAds ～ 致力于构建优质的 Flutter 广告变现插件

## 🎯 产品里程碑

- 🙏 感恩你我有缘在此相遇。
- 💻 从 18 年我开始学习投入 Flutter 企业级项目研发，目前负责百万级日活 App 的研发和广告运营。
- 🎉 20 年发现社区没有成熟广告变现插件，开发并开源了穿山甲和优量汇插件(flutter_pangle_ads、flutter_qq_ads)，目前收获了近 500 Star。
- 🚀 21 年随着对广告运营深入学习，又打造了聚合广告插件(flutter_gromore_ads) 来获得更高的广告收益，并将基础版本做了开源给社区开发者使用。
- 🎁 23 年不断对广告模式和业务场景的探索，以及广告深入指标的精细化运营，过程中不断迭代插件功能(测试助手、实时价格、超前加载、广告监控)，推出了 GroMore Pro 版本.
- 🎬 24 年短剧和小视频已近比较成熟，但基础成本还比较高（版权、拍摄、存储、带宽、运营等），穿山甲内容输出免费提供，于是开发了短剧小视频插件帮助开发者从功能变现拓展到内容变现。
