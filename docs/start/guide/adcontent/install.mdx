# 📦 集成安装

## 介绍

- 短剧小视频的插件，是将今日头条巨量引擎内的海量内容结合特有的技术优势,同时利用穿山甲变现优势与穿山甲广告打通，以内容+广告一体化的形式输出给开发者。
- 帮助开发者提升用户活跃时长和用户留存，并带来广告变现收益。
- 2022年付费短剧呈现爆发式增长，消耗量级持续突破新高，免费短剧可采用激励广告解锁剧集的形式进行，用激励视频进行广告变现。

## 核心功能
- ✅ 初始化（与 [GroMore Pro 版](start/guide/gromore/install)完全兼容，其他广告插件可能不兼容）
- ✅ 打开短剧聚合页面
- ✅ 打开短剧详情页面
- ✅ 打开短剧滑滑流（混排）页面
- ✅ 获取所有短剧
- ✅ 获取个性化推荐短剧
- ✅ 获取短剧详情根据短剧 id 
- ✅ 获取短剧分类
- ✅ 获取短剧列表根据分类
- ✅ 搜索短剧
- ✅ 校验短剧参数正确性
- ✅ 获取观看历史记录
- ✅ 清除观看历史纪录
- [ ] 支持[微信]和[支付宝] 支付功能（开发中，后续Pro 版可免费更新）

- 📱 支持 Flutter 页面自定义短剧列表和搜索等功能
- 📦 与[GroMore Pro 版](start/guide/gromore/install)完全兼容
- 📦 与[AdSaprk Pro 版](start/guide/adspark/install)完全兼容



## 通过本地安装插件

:::tip{title="Pro 版安装"}
购买后我将提供 'flutter_adcontent.zip' 压缩包，在项目根目录下新建一个 'plugins' 文件夹，然后将压缩包解压至此
:::

```yaml title="pubspec.yaml"
dependencies:
  flutter_adcontent:
    path: plugins/flutter_adcontent # 本地引入
```
