# 📱 小视频详情

## 介绍

小视频提供 2 种模式方式进行展示

- Widget 组件方式可嵌入 Flutter 页面，自由度高
- 打开原生页面，自由度有限

## 小视频组件

``` dart title="example/pages/video_page.dart"
/// [width] View 的宽度
/// [height] View 的高度

/// [bottomOffset] 内容距底部的距离

/// [contentType] 展示内容类别
```

## 播放控制器

``` dart {2,6,13,20-21}title="example/pages/drama_detail.dart"
/// 在 initState 中初始化，并传递给组件
late ProVideoController controller;

@override
void initState() {
  controller = ProVideoController();
}

@override
Widget build(BuildContext context) {
  return Scaffold(
      body: DramaWidget(
        controller: controller,
        ...
        )
  )
}

/// 暂停&继续
controller.pause();
controller.resume();
```

## 打开小视频原生页面

``` dart {4,13,22} title="example/pages/my_page.dart"
/// 基本参数与 Widget 基本一致
// 打开全屏视频
FlutterAdcontent.showVideoPage(
      style: VideoDrawParams.drawStyleFullscreen,
    );

// 打开宫格模式
FlutterAdcontent.showVideoPage(
      hideClose: true,
      hideChannelName: true,
      hideFollow: true,
      channelType: VideoDrawParams.drawChannelTypeRecommend,
      style: VideoDrawParams.drawStyleGrid,
    );

// 打开双 Feed 流模式
FlutterAdcontent.showVideoPage(
      hideClose: false,
      hideChannelName: true,
      hideFollow: true,
      channelType: VideoDrawParams.drawChannelTypeRecommend,
      style: VideoDrawParams.drawStyleDoubleFeed,
    );
```
