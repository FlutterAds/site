# 📽 历史纪录

## 介绍

获取到历史纪录是数据列表形式，可自定义页面布局（如：在个人中心直接展示观看历史纪录），Flutter 层业务逻辑相同仅需一套代码实现高度自定义。

## 获取观看历史记录

``` dart
/// [page] 页数，从1开始，小于1时全量获取
/// [count] 每页个数，小于1时全量获取
List<Drama> list = await FlutterAdcontent.getDramaHistory();
```

## 清除观看历史纪录

``` dart
bool result = await FlutterAdcontent.clearDramaHistory();
```
