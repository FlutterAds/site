# 📦 集成安装

:::tip{title="购买 Pro 版，请微信与我联系"}
购买后我将提供 'flutter_gromore_ads.zip' 压缩包，在项目根目录下新建一个 'plugins' 文件夹，让后将压缩包解压至此
:::

## 方式1：通过本地安装插件

```yaml title="pubspec.yaml"
dependencies:
  flutter_gromore_ads: 
    path: plugins/flutter_gromore_ads # 本地引入
```

## 方式2：通过 Git 安装插件

:::tip {title="购买 Pro 版，请微信与我联系"}
购买后将我提供的 'flutter_gromore_ads.zip' 压缩包，解压并上传到自己的 'git' 仓库后，可通过此方式引入
:::

```yaml title="pubspec.yaml" {4}
dependencies:
  flutter_gromore_ads:
    git:
        url: 这里写自己的 git 仓库链接
```

## 方式3：开源版安装插件

:::tip {title="购买 Pro 版，请微信与我联系"}
开源版可直接通过 pub.dev 安装，与其他插件一样
:::

```yaml title="pubspec.yaml" {4}
dependencies:
  flutter_gromore_ads: ^2.1.0
```
