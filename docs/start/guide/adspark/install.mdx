# 📦 集成安装

## 介绍

- AdSpark 的广告监测插件，提供全渠道广告监测服务。
- 简单来说就是投流推广花了多少钱，安装、激活、看广告、领积分、充值的有多少？最后整体带来多少收益的全链路全方位监测。
- 本插件仅提供 Pro 版本，不开源。


## 支持平台

穿山甲、巨量引擎、巨量广告

## 核心功能
- ✅ 激活
- ✅ 注册
- ✅ 支付
- ✅ 账户登录、登出
- ✅ 绑定社交账号
- ✅ 完成教学任务/副本
- ✅ 升级
- ✅ 创建角色
- ✅ 查看内容/商品详情
- ✅ 添加到购物车
- ✅ 提交购买/下单
- ✅ 添加支付渠道
- ✅ 数据检测
- ✅ 自定义事件
- 📱 可视化测试助手

## 通过本地安装插件

:::tip{title="Pro 版安装"}
购买后我将提供 'flutter_adspark.zip' 压缩包，在项目根目录下新建一个 'plugins' 文件夹，然后将压缩包解压至此
:::

```yaml title="pubspec.yaml"
dependencies:
  flutter_adspark: 
    path: plugins/flutter_adspark # 本地引入
```
