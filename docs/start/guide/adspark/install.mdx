## 引入插件

:::tip{title="Pro 版安装"}
购买后我将提供 'flutter_adspark.zip' 压缩包，在项目根目录下新建一个 'plugins' 文件夹，然后将压缩包解压至此
:::

```yaml title="pubspec.yaml"
dependencies:
  flutter_adspark: 
    path: plugins/flutter_adspark # 本地引入
```
