## 初始化

```dart title="example/main.dart"
// 导入引用
import 'package:flutter_adspark/flutter_adspark.dart';

/// [appId] 应用ID
/// [channel] 渠道
/// [autoTrackEnabled] 是否开启自动采集
bool? result= await FlutterAdspark.init("appid", "huawei", true);
```

## 事件上报

> 激活事件由 SDK 初始化成功后自动触发，不需要自己上报

### 系统事件

``` dart
/// 注册事件（系统）
/// [method] 注册方式
/// [isSuccess] 是否成功
FlutterAdspark.registerEvent("wechat", true);
```

### 个性事件

``` dart
/// [eventName] 事件名称
/// [params] 参数信息
FlutterAdspark.addEvent(GrownEventType.order, params: {
                        "orderId": "54323454323",
                        "amount": 2000,
                        "count": 1,
                      });
```
