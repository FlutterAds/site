# 🏷️ 横幅广告(Banner)

:::tip{title="💡 小提示"}

 - 宽高设置比例按照在 GroMore 后台的配置来计算
 - 推荐使用信息流，Banner 收益最低，广告形式落后，逐渐会被信息流取代
 - [参考官方文档](https://www.csjplatform.com/union/media/union/download/detail?id=142&docId=27627&osType=android)
 - Pro 版客户，如有问题随时联系我，提供配置指导

:::

```dart title="example/pages/banner_page.dart"
/// 横幅广告也叫 Banner 广告
/// [posId] 广告位 id
/// [width] 宽度
/// [height] 高度
/// [show] 是否显示广告
AdBannerWidget(
    posId: posId,
    width: 300,
    height: 75,
    show: true,
)
```
