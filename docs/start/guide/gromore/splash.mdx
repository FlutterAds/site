## 全开屏

```dart title="example/pages/splash_page.dart"
/// [posId] 广告位 id
/// [timeout] 加载超时时间
FlutterGromoreAds.showSplashAd(
  posId,
  timeout: 4.5,
);
```

## 半开屏

```dart title="example/pages/splash_page.dart"
/// [posId] 广告位 id
/// [logo] 底部 logo 资源名称
FlutterGromoreAds.showSplashAd(
  posId,
  logo: 'flutterads_logo',
);
```

> Logo 图配置请参考示例
