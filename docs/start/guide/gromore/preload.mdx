## 广告预缓存

```dart title="example/pages/home_page.dart"
/// 初始化完毕后，就可以进行预缓存了
/// [rewardPosids] 激励视频广告位 id 列表
/// [insertPosids] 插屏广告位 id 列表
/// [concurrent] 并行加载的广告位数，默认值为2，合法值为[1,20]
/// [interval] 时间间隔，默认值为2s，合法值为[1,10]
/// [customData] 激励视频-设置服务端验证的自定义信息
/// [userId] 激励视频-设置服务端验证的用户信息

FlutterGromoreAds.preload(
      rewardPosids: rewardPosids,
      insertPosids: insertPosids,
      customData: 'customData',
      userId: 'userId',
      concurrent: 2,
      interval: 2,
    );
```
