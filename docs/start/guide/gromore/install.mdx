# 📦 集成安装


## Pro 版安装插件

:::tip{title="Pro 版安装"}
购买后我将提供 'flutter_gromore_ads.zip' 压缩包，在项目根目录下新建一个 'plugins' 文件夹，然后将压缩包解压至此
:::

```yaml title="pubspec.yaml"
dependencies:
  flutter_gromore_ads: 
    path: plugins/flutter_gromore_ads # 本地引入
```

## 开源版安装插件

:::tip {title="开源版，不提供服务支持，多看文档"}

- 开源版可直接通过 pub.dev 安装，与其他插件一样
- [开源版与 Pro 版本对比](/start/vs)

:::

```yaml title="pubspec.yaml"
dependencies:
  flutter_gromore_ads: ^2.1.0
```
