# 📱 插屏广告

:::tip{title="💡 小提示"}

 - `全插屏、半插屏、横插屏`都是在 GroMore 后台配置
 - Pro 版客户，如有问题随时联系我，提供配置指导

:::

## 全插屏

```dart title="example/pages/fullscreen_video_page.dart"
/// 就是全屏视频广告
/// [posId] 广告位 id
FlutterGromoreAds.showInterstitialAd(posId);
```

## 半插屏

```dart title="example/pages/fullscreen_video_page.dart"
/// 类似于弹窗广告
/// [posId] 广告位 id
FlutterGromoreAds.showInterstitialAd(posId);
```

## 横插屏

```dart title="example/pages/fullscreen_video_page.dart"
/// 这里要确保手机是横屏的，可旋转横屏的
/// [posId] 广告位 id
FlutterGromoreAds.showInterstitialAd(posId);
```
