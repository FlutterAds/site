# 🙋 常见问题

## Pro 版购买后是永久使用吗？

:::tip{title="💡"}

是的，Pro 版提供永久免费更新使用，可多个项目使用，开源版更新速度慢于 Pro 版。

:::

## GroMore 支持哪些广告平台和广告类型？

:::tip{title="💡"}

<img src={normalizeImagePath('/gromore_ads_list.png')}/>

- 支持广告平台：穿山甲、优量汇、百度、快手、AdMob、SigmobAd、UnityAds、Mintegral，参考【[GroMore 官网](https://www.csjplatform.com/gromore)】
- 支持广告类型：开屏、插屏、全屏视频、激励视频、Banner、信息流，参考【[GroMore 官网](https://www.csjplatform.com/basic-monetization)】


:::

:::tip{title="💡"}

目前支持穿山甲、优量汇插件是完全开源的，GroMore 穿山甲平台旗下的聚合广告插件分为开原版和 Pro 版本

|-|-|-|
|[flutter_gromore_pro](https://flutterads.top/)|<Badge text="🚀 Pro" type="warning" />|穿山甲、GroMore 聚合 Flutter 广告插件，支持主流广告平台，可以通过竞价提升议价权，运营提高广告价值|
|[flutter_adcontent_pro](https://flutterads.top/)|<Badge text="🚀 Pro" type="warning" />|穿山甲短剧小视频，内容输出 Flutter 插件，汇集了近 4000 部优质短剧资源和无限的抖音短视频资源，可直接免费使用|
|[flutter_adspark_pro](https://flutterads.top/)|<Badge text="🚀 Pro" type="warning" />|穿山甲、巨量引擎，广告监测、广告归因 Flutter 插件，投流推广必备，快速计算 ROI 和 LTV 等指标|

|[flutter_gromore_ads](https://github.com/FlutterAds/flutter_gromore_ads)|✅| 字节跳动、GroMore 聚合 Flutter 广告插件|

## Pro 版不会集成使用，接入遇到问题怎么办？

:::tip{title="💡"}

- 遇到问题可以第一时间给我反馈，我发给你问题对应的参考文档。
- 如果还是解决不了咱们可以约时间远程语音提供技术支持，帮你解决问题。
- 遇到任何广告插件的问题，我都会全程协助解决。
- 开源版仅提供 [issues ](https://github.com/FlutterAds/flutter_gromore_ads/issues)解答，不提供支持服务。

:::

## 开源版可以快速切换到 Pro 版吗？

:::tip{title="💡"}

与开源版相比，Pro 版本有更多的广告接口能力，底层基础框架是差不多的，Flutter 接口调用层可实现无缝切换，购买后替换一下插件依赖方式即可。

:::

## 我想实现 xxx 功能可以帮我做吗？

:::tip{title="💡"}

- 如果是官方的广告接口 Pro 版还未支持，可以免费支持，并且会将此接口免费开放给其他 Pro 版客户使用。
- 如果是你的业务功能开发，需要单独聊具体需求单独收费。

:::

## Pro 版购买后我想退款可以吗？

:::tip{title="💡"}

因是源码包产品具有非常强的传播性，我给您提供了源码包后将无法退款。

:::
