## 1、行业大盘CPM波动趋势

- 11月1日-11月11日：受双十一大促影响整体cpm较高，其中11月9日-11月11日为大促核心冲量节点，行业cpm达到峰值；
- 11月12日-11月30日：双十一大促结束电商预算退坡，cpm呈下降趋势。
